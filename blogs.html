Question-1: What are the different ways to select an element in the DOM?

To select an element in the DOM we can use getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

-innerHTML can read both the HTML markup and the text content of an element.
-innerText can read the content of an element and it returns the text exactly it displays.
-textContent only deals with the raw text and it's not respnsible for styles.

Question-3: What is event delegation in the DOM?

It's a technique where we can connect a single event listener to a parent element, letting it to handle events triggered on its child elements, rather than adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

When an event occurs on an element, it triggers not only on that element but also propagates upwards through the DOM hierarchy, triggering event handlers on its parent elements until it reaches the document's root.
